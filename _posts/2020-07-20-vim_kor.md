## 리눅스 VIM 에서 한글깨짐현상 해결하기
---

>vim을 사용하다보면 간혹 한글이 제대로 나오지 않는 경우가 있습니다.
>UTF-8 인코딩문제로 아래와 같은 방법으로 해결할 수 있습니다.

<br>

### 1. 내 계정에만 적용.

```
$ ~/.vimrc
```
<br>

### 2. 전체사용자 계정에 적용.

```
$ /etc/vim/vimrc
```
<br>

### 3. 아래내용을 위파일에 추가.

```
set encoding=utf-8
set fileencodings=utf-8,cp949
```
