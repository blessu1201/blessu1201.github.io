> vim을 사용하다보면 간혹 한글이 제대로 나오지 않는 경우가 있습니다.  
> UTF-8 인코딩문제로 아래와 같은 방법으로 해결할 수 있습니다.

<br>

## 1. 아래내용을 파일(.vimrc 또는 vimrc)에 추가.

파일위치는 아래를 참고하세요.

```bash
set encoding=utf-8
set fileencodings=utf-8,cp949
```
<br>

## 2. 내 계정에만 적용.

```bash
$ ~/.vimrc
```
<br>

## 3. 전체사용자 계정에 적용.

```bash
$ /etc/vim/vimrc
```
